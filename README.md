# C2_alpha

command and control via http

# Requirements

* Python 3.6
* Flask
* Flask Auto-Index
* Cmd2

# Features

* Backend 
    * directory listing
* Shell cmd2
    * List bots
    * Send command to the bots 
    * Se the log of the command's results
